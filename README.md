# lab2_repo

Проект для лабораторной работы №2 по работе с Git и GitHub.

## Описание проекта

Простой проект на Python, демонстрирующий основные функции работы с Git:
- Создание репозитория
- Работа с ветками
- Разрешение конфликтов
- Создание релизов

## Функциональность

- Вывод приветственного сообщения
- Функция сложения двух чисел
- Функция умножения двух чисел

## Установка и запуск

### Предварительные требования
- Python 3.x
- Git

### Запуск проекта
```bash
# Клонировать репозиторий
git clone https://github.com/WEBTERR0R/lab2_repo

# Перейти в папку проекта
cd lab2-repo

# Запустить программу
python src/main.py
